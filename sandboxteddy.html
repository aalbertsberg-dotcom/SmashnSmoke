01 // WEEKLY SPECIALS

Auto-updates on site

Edit the special name and description for any day. Hit Generate Code to get the JS snippet — paste it into the weeklyOverrides block in index.html to override specific dates, or edit the specialMap object to change the permanent rotation. 


      
    

  


  
  

    

      
02 // SITE MAP

      Dev Notes
    

    

      

        
Key Sections

        
    
          
  • #hero — Logo, tagline, today's special badge, CTA buttons
    • 
          
  • #specials — Weekly rotation grid, day cards
    • 
          
  • #menu — Three-panel menu board (Wings, Burgers, Combos)
    • 
          
  • #partner — Butcher on Tucker section
    • 
          
  • #find — Location, hours, social links, map placeholder
    • 
          
  • #about — Tracey's story, LOKÉ explainer
    • 
          
  • footer — Links, Double-A credit, partner strip
    • 
        

      

      

        
JS Hooks

        
    
          
  • weeklyOverrides — Date-specific special overrides. Format: 'YYYY-MM-DD': ['Name','Desc']
    • 
          
  • specialMap — Default day-of-week rotation
    • 
          
  • .reveal — Scroll animation class, handled by IntersectionObserver
    • 
          
  • .today — Auto-added to today's special card in the grid
    • 
          
  • .special-text — Hero badge text, updated by JS on load
    •

    • 
          
  • Photo gallery — Tracey will need food photos eventually
    •

    • 
          
  • Mac N Cheese as a rotating special — add to specialMap

File Structure

        
    
          
  • index.html — Entire site, single file
    • 
          
  • sandboxteddy.html — This page
    • 
          
  • Logo is base64-embedded in index.html — swap for external file when optimizing load time
    • 
          
  • Fonts loaded from Google Fonts CDN
    • 
          
  • No frameworks, no dependencies — pure HTML/CSS/JS
